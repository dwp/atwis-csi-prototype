Adviser Decision Letters

Introduction

The latest revisions, valid from the 12th April 2018 are available. These will need to be used in conjunction with DWP guidelines see below for further information.

Each of the letters has a corresponding set of completion notes to tell you what should be included in each section.

Use the letters by:

  • opening up a current intranet copy and saving a copy to the customer folder located in the shared folder (This will allow you to update it with your customer’s information and print copies)
  • Always download a letter from the intranet and NOT a saved copy, as letters may be updated regularly. Existing letters should not under any circumstances be amended.or used to produce other letters.

Our letter templates are editable and have been designed to add details and amend as required. For any technical problems, please check the operations guidance (Completing interactive Word letters) before emailing DWP OED Channel Production Team
